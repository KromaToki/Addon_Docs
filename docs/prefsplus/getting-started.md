# Getting Started

## Install and Open PrefsPlus

1. Install the PrefsPlus ZIP or source folder through Blender's add-on workflow.
2. Enable PrefsPlus in Edit > Preferences > Add-ons.
3. Open the PrefsPlus add-on preferences.
4. Expand PrefsPlus Configuration.
5. Click Add To Header.
6. Search for an installed add-on and select it.

The new button appears in the Preferences header.

## Basic Workflow

1. Add one or more add-ons to the header.
2. Click a header button to switch to Preferences > Add-ons and filter to that add-on.
3. Click the same active button again to clear the filter.
4. Edit each pin's label, icon, visibility, and order from the PrefsPlus preferences.
5. Save a profile when the layout is ready.

## Header Button Behavior

PrefsPlus pins are visible in the Preferences header when the Add-ons section is active.

If Persistent Header Buttons is enabled, pins stay visible in other Preferences sections too.

PrefsPlus also adds a Preferences button for PrefsPlus itself at the end of the pin list.

## Compatibility Notes

- PrefsPlus requires Blender 4.5 or newer.
- The section dropdown controls require Blender 5.0 or newer.
- The compact filtered add-on view is optional and can be disabled independently from header pins.
