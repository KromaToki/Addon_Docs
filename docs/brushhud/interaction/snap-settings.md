Snap Settings define the stepped values used when precision modifiers are active during BrushHud adjustments.

If you want predictable, repeatable increments instead of continuous motion, this is the page to tune.

## Size Adjustments

### Discrete Step

Discrete Step sets the step size used when size snapping is active.

- Increase it for bigger jumps.
- Reduce it for finer stepped control.

### Alt+Drag Snaps to Integer

Alt+Drag Snaps to Integer forces size adjustments to land on whole-number values.

- Enable it when you want clean integer brush sizes.
- Leave it off if fractional sizes are useful to your workflow.

## Strength Adjustments

### Discrete Step

Discrete Step sets the strength increment used when strength snapping is active.

- Increase it for broader stepped strength changes.
- Reduce it for more granular control.

### Alt+Drag Snaps to Integer

Alt+Drag Snaps to Integer makes strength land on rounded stepped values instead of flowing continuously.

- Enable it if you want more predictable repeated strength values.
- Leave it off if you prefer fully continuous drag response.

## Rotation Adjustments

### Discrete Step

Discrete Step sets the rotation increment used when rotation snapping is active.

- Use lower values for fine angular control.
- Use higher values for deliberate stepped rotation.

### Sensitivity Modifier

Sensitivity Modifier changes the rotation step when the slow or precision modifier is held.

- Use it to create a second, finer snap scale without changing the base step.
- This is useful when you want both fast coarse rotation and precise alignment.

### Alt+Drag Snaps to Integer

Alt+Drag Snaps to Integer rounds rotation to whole-degree changes.

- Enable it when you want clean degree values.
- Leave it off if you need freer angle control.
