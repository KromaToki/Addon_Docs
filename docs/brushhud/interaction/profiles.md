Interaction profiles store the behavioral side of BrushHud. They are useful when you want different trigger setups, snapping preferences, or drag behavior for different workflows.

## Where This UI Lives

The Interaction profile controls appear at the top of the main Interaction area, before the detailed settings pages.

## Profile Menu

The profile menu chooses which saved Interaction profile is currently active.

- Use it to switch between existing input setups.
- If no saved profiles exist yet, BrushHud shows a placeholder instead of a selection menu.

## Save

Save updates the currently selected Interaction profile with your current settings.

- Use this when you have refined an existing setup and want to keep the same profile name.
- It is the fastest way to preserve small adjustments.

## Delete

Delete removes the selected Interaction profile.

- Use this when a preset is obsolete or duplicated.
- Delete is only meaningful once profiles already exist.

## Save As

Save As writes the current Interaction settings to a new profile name.

- Use this when you want to branch from a working setup without overwriting it.
- This is usually the safest save action for experimentation.

## Import

Import brings an Interaction profile in from a file.

- Use it to load a preset shared from another machine or another user.
- Imported profiles become part of your local BrushHud profile list.

## Export

Export writes the current or selected Interaction profile to a file.

- Use it for backup, transfer, or sharing.
- Export is the cleanest way to move behavior settings between systems.

## Open Folder

Open Folder jumps to the directory where Interaction profiles are stored.

- Use it when you want to manage the files directly.
- This is also useful when you need to manually copy or archive presets.

## When to Start Using Profiles

You do not need profiles to begin using BrushHud. They become valuable once one of these is true:

- you have more than one preferred input style,
- you switch between different devices or gestures,
- you want a backup before major interaction changes.
