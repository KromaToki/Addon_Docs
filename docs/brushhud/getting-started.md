# Getting Started

## Install and Open the Preferences

1. Install the BrushHud zip in Blender through Edit > Preferences > Add-ons.
2. Enable the addon.
3. Open the BrushHud preferences.

## Full vs Core

This manual is written against the 'Full' version.

- If a control is marked with `*`, it is Full only.


## Profiles

BrushHud has profile tools at the top of both main sections: Interaction and Graphics. Once you have a setup you want to keep, export, or reuse, save them!

- Interaction profiles store how the addon behaves.
- Graphics profiles store how the HUD looks.


## Recommended First Setup Path

If you are configuring BrushHud for the first time, do the following in order:

1. Set up the trigger keys in [Interaction: Key Bindings](./interaction/key-bindings.md).
2. Adjust movement feel in [Interaction: General Behavior](./interaction/general-behavior.md).
3. Set snap increments in [Interaction: Snap Settings](./interaction/snap-settings.md).
4. Choose or save a graphics profile in [Graphics: Profiles and Themes](./graphics/profiles-and-themes.md).
5. Tune the most visible HUD elements in [Graphics: Size](./graphics/size.md) and [Graphics: Strength](./graphics/strength.md).
