# BrushHud Manual

### Overview
BrushHud adds enhanced Graphics and interactions for brush size, strength, rotation.


!!! danger "Important Limitation"
      Brushhud can't display Blender's procedural textures such as cloud or perlin (among others). Image files (such as jpg, png, etc.) are good to go.

    ### Full version: Changes to the graphics settings must be manually saved after editing for (future blender sessions). 

    ### Platform Support
    Works with Blender 4.5 and up (tested on windows)

## BrushHud Versions
BrushHud has two variants: 
  - Brushhud 'Full' (gumroad link)
  - Brushhud Core (free).

| Feature | BrushHud (Full) | BrushHud (Core) |
| :--- | :---: | :---: |
| Interaction Features| ✅ | ✅ |
| Graphics HUDs | ✅ | ✅ |
| Graphics Editing | ✅ | partial |
| Profiles I/O  | ✅ | - |
| HUD Preview| ✅ | - |
| HUD Color Editing  | ✅ | - |
| Additional Profiles | ✅ | - |

### Full Version (Available Soon): 


  - Graphics: all visual features of the HUD can be tuned, or color matched to Blender themes.
  - Import/Export and share graphics profiles
  - Interaction: Multiple ways to trigger the features, and control how the HUD behaves
  - HUD Previewer: Automatically Runs BrushHud in the 3d viewport as you adjust colors in the property editor.
  - Hud Component Color editing such as fills, lines, and text 
  - Includes many (TBD) graphics profiles matched to popular Blender themes.
  - Includes 'Inverto X' a custom Blender theme and matching graphics profile

### Core Version


  - All Interaction properties that the Full version has
  - Most Graphics adjustments, toggling visual features
  - HUD preview, color editing, and import/export of graphics profiles not available
  - Ships with HUD profiles matched to Blender's default Light/Dark themes
  - Includes custom Blender Dark and Light themes (minor adjustments)
  - limited graphics profile features.
